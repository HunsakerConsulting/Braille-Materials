Section 2.7 Cut, Copy, and Paste

Sometimes when editing a document it is necessary to delete or move text around. When you "cut" text to the clipboard, you remove it from the document and place it on the clipboard. The clipboard is an invisible area where text or graphics are stored. Information stays on the clipboard until you paste it somewhere else or until you place other information on the clipboard.

To "copy" text to the clipboard means that you leave one copy in the original location and move another copy to the clipboard. You may paste text from the clipboard to the position where your cursor is located. Use the commands below to practice copying, cutting, and pasting in a document that is for practice.

Subsection 2.7.1 Quiz

  • Explain the difference between "copying text" and "cutting text".

  • What is the clipboard?

  • Where is information "pasted" from the clipboard to?

  • Describe a situation in which you may need the clipboard.

.

Subsection 2.7.2 Tasks

Practice the commands listed here.

  • Open a practice document. Write some numbered sentences.

  • Select the entire document. Use CONTROL + A.

  • Cut the text to the clipboard. Use CONTROL + X.

  • The information is now on the clipboard, but not in your document. Paste the text back in your document by using CONTROL + V.

  • Select all text again. Copy it to the clipboard by using CONTROL + C.

  • Jump to the end of the document. Press enter twice. Paste the text from the copy on the clipboard. Press CONTROL + V. Now you have two copies of the same text.

  • You can select by line. Use SHIFT + up ARROW or SHIFT + down ARROW.

  • Select text by character with SHIFT + right ARROW or SHIFT + left ARROW.

  • CONTROL + shift + right ARROW will select by word to the right. CONTROL + shift + left ARROW will select by word to the left.

  • SHIFT + END will select from the cursor to the end of a line. SHIFT + HOME will select from the cursor to the beginning of a line.

Practice using the select, cut, copy, and paste commands to move text around in a document. Text will remain on the clipboard until it is replace by something else. You can use the DELETE or BACKSPACE key to delete selected text.
