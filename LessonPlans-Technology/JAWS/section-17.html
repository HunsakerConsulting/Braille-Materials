Section 2.13 Create a Shortcut Command

Often to open an application we go to the desktop, click on an icon, and open the program. This can be done for a program such as Microsoft Edge, Chrome, or Firefox. It is possible to create your own keyboard command that will allow you to open an application. In this way, you can open the program from anywhere on your PC. You don't have to return to the desktop or go to the Start Menu to open Programs. Simply press the keyboard shortcut command you create.

You can create a short cut to open almost any program. Make a shortcut to open your e-mail application (such as Outlook, Thunderbird, etc.), internet browsers (such as Microsoft Edge, Chrome, or Firefox), or Microsoft Word.

Subsection 2.13.2 Tasks

Complete the steps below to create a shortcut command that will allow you to open Microsoft Edge, Chrome, or Firefox using a shortcut command.

  • Use WINDOWS KEY + D to bring focus to the desktop.

  • Press letter I until Microsoft Edge, Chrome, or Firefox is the focus.

  • Press ALT + ENTER to open the properties dialog box. Another method would be to right click on the icon. You can press SHIFT + F10 to right click. Then DOWN ARROW to "Properties" and press ENTER.

  • TAB to the "shortcut key" field.

  • Press the letter I.

  • The shortcut will be set to CONTROL + alt + I to open Microsoft Edge, Chrome, or Firefox.

  • TAB to continue through the dialog box.

  • At the "run" combo box, you can use UP ARROW or DOWN ARROW to select normal, minimized, or maximized. If you choose maximized, the browser will always start "maximized".

  • Press ENTER when done making changes.

  • Now when you press CONTROL + alt + I, whatever browser you selected should run. Try it out!}
