Section 2.2 Computer or My Computer

In the start menu there is a "Computer" or "My Computer" item. On some computers, there is a desktop icon called "Computer" or "My Computer". Press the windows KEY to open the start menu tab to the right column of the start menu. Down ARROW to "computer" and press ENTER. There are many ways to access the "My Computer" area of your PC.

In "My Computer" you can access different computer drives. The "c" drive is the hard drive of your computer. That is where programs and other files are stored. The "d" drive is usually a compact disk player. The name of the CD will show up when one is installed on your computer. If you are using a thumb-drive, drive "e" or "f" is usually where this will show up. To access information on different drives, you will need to know how to use "Computer" or "My Computer".

Subsection 2.2.2 Tasks

Using keyboard commands can save time. You can keep your hands on the keyboard without using the mouse. Practice these shortcut commands. These are Windows commands, not JAWS commands.

  • Open My Computer or Computer from the Start Menu or desktop

  • Press INSERT + T to have JAWS say the window title.

  • In a list of folders or drives, use the initial letter of an item to jump to it. Try this in "Computer" or "My Computer:.

  • Use CONTROL + HOME to jump to the top of a list.

  • CONTROL + END will jump to the bottom of a list.

  • Alt + left ARROW will move back to the previous open window.

  • CONTROL + W will close an open window.

  • Windows key + E will open the "Computer" or "My Computer" system. This is known as Windows Explorer.

  • To have JAWS speak the date, press INSERT + F12 twice quickly.
