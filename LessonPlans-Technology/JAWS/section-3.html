Section 1.3 JAWS Start-Up Wizard

When JAWS is installed, a start-up wizard will run. You can access this wizard at any time to manipulate JAWS settings. The speech rate, punctuation level, and typing echo can be changed to meet your preferences. The typing echo is what you hear when you keyboard.

There are other settings in the JAWS start-up wizard that can be altered. You can control how JAWS is started. If you do not always use JAWS, you can set the option to have JAWS never run automatically. In this case, JAWS would need to be turned on by using a hotkey, desktop icon, or by going through the start menu.

In the wizard, you can set JAWS to run at the log in screen or after the log in screen.

Other options can be changed by using the JAWS start-up wizard. Keyboard, braille,and verbosity options can be modified. At any time when using JAWS, you can get help. INSERT + F1 will give help. After listening to information gained from INSERT + F1 help, use ESCAPE to close help.

Subsection 1.3.2 Tasks

Learn to use the JAWS start-up wizard to quickly change options to meet your preferences. Follow the steps below to start the wizard:

  • Press ALT + TAB to switch to the JAWS window. If you have JAWS running, this JAWS window will also be running.

  • Press ALT + H to open the help menu.

  • Press letter Z to open the startup wizard.

Complete these steps with your student. Use the start-up wizard to make changes and start learning what your preferences are.

  • The first option allows you to install the tutorial from the JAWS CD. Follow the instructions given to do this. Older students would greatly benefit from listening to training materials from Freedom Scientific. It is also suggested that instructors listen to the tutorials provided in the Training section of the "JAWS help" menu.

  • If you have already installed the tutorial, press ENTER to go to the next screen.

    • In the dialog box that appears, you can control the JAWS speech settings. You can adjust the voice rate, the punctuation, and typing echo.

    • Adjust the voice rate by moving the slider bar. Use left ARROW or right ARROW to adjust the speaking rate. Page UP or page DOWN will allow you to move in larger chunks.

    • When done changing the voice rate, press TAB to move forward through the dialog box. SHIFT + TAB will move back through the dialog box if needed.

    • Next you can control how much punctuation is spoken. Use ARROW KEYS to make choices in the combo box. TAB when finished.

    • TAB again. You will be taken to a group of buttons. These buttons are to change typing echo. Typing echo is what you hear when you type.

    • Use UP ARROW or DOWN ARROW to select the button you want.

    • At any time, get help by pressing INSERT + F1. Try this frequently when you are first using JAWS. ESCAPE to close the "help" window.

      • Try getting help when on the key echo radio buttons.

      • In the help window, use UP ARROW or DOWN ARROW to read. ESCAPE to close the help window.

    • Enter to go to the next screen in the wizard.

    • The screen will allow you to change when JAWS is started.

  • Learn about what each option is..

    1. If you set JAWS to load before the log in screen, speech will be available when logging in with your name and password.

    2. You may set the option for JAWS to start running after log in for all users. In this way, JAWS would always be running for you when you want to use speech with your PC.

    3. You can set JAWS to start after log in for "only this user". How you set JAWS to run depends on your individual situation. If you are the only person using your computer, then these options won't be quite as important to you.

  • ENTER to go to the next screen.

  • Keyboard settings allow you to control use of a Virtual Ribbon Menu, Keyboard Layout, and Initial state of num lock key.

  • JAWS provides a virtual ribbon to make access to menu options user friendly. The virtual ribbon is similar to more traditional ribbons used by computers running earlier versions of Windows.

Alter Keyboard Layouts.

  • You can control which keyboard layouts that you use with JAWS. For example, you can choose between the laptop or desktop keyboard layout. You should have explored this in an earlier lesson when we changed the keyboard layout to laptop.

  • The use of NUMLOCK can be controlled. We'll leave it off for now. If you want to learn more about this, use INSERT + F1 to get help.

  • Verbosity settings are on the next screen.

  • Verbosity settings include tutor messages, access keys, verbosity level, and graphic verbosity. Verbosity is the amount of help JAWS will automatically give you.

  • Use ARROW KEYS to choose. Get help as needed.

  • There is also a Braille screen. You may not be starting with braille. SPACE on the Finish button to exit the wizard. Follow any instructions your PC gives.
