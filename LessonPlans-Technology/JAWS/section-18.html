Section 2.14 JAWS and Reading a Web page

As with most tasks, a screen reader such as JAWS can help you read and navigate web pages more efficiently. TAB is used to move forward from link to link. SHIFT + TAB will move back by link. To follow a link, press ENTER.

When on web pages, there are special JAWS commands to get lists of certain elements. An element is an item such as a link, frame, or heading. Using these lists can make it even faster to navigate web pages. When using JAWS on a web page, you can work in Reading Mode or Forms Mode.

On a web page or anywhere, have JAWS read the status line. Press insert + downarrow to get the information on the status line. To say all, press insert + down arrow. Try this in various computer applications as well as when on the Internet. Control will stop JAWS from reading as needed.

Subsection 2.14.2 Tasks

  • Bring up the start menu.

  • Type in a URL like "www.ksl.com" and press ENTER.

  • JAWS will announce the number of headings and links. Then JAWS will start reading the page. You may press CONTROL to stop JAWS from reading.

  • Use UP ARROW or DOWN ARROW to read back and forward by line. Explore the page using UP ARROW or DOWN ARROW.

  • Move to the top of the page. Use CONTROL + HOME.

  • Press INSERT + T to read the page title.

  • Get help. Press INSERT + F1. You will get a quick tip and will be told about the number of links, headings, and forms on the page. Use UP ARROW or DOWN ARROW to read. ESCAPE to close help. Use INSERT + F1 to let JAWS help you learn how to navigate web pages and complete other tasks.

  • On a web page, you can move forward by page heading by pressing letter H. SHIFT + H will jump back by page heading. Try this on the web page.

  • Letter P will let you jump by paragraph. SHIFT + P will move back by paragraph.

  • You can get a list of links. Press INSERT + F7 to bring up the links dialog box. Up ARROW or DOWN ARROW through the list of links. Press ENTER to follow the link.

  • Insert + H will open the headings dialog box. use ARROW KEYS to cycle through the list of headings. Press ENTER to go to a heading.

  • To go back to the previous web page, use ALT + left ARROW. ALT + RIGHT ARROW will take you forward by page in your current session.

  • Explore as time allows. ALT + F4 to exit when you finish.

Use Surf's Up:

  • Press INSERT + J to open the JAWS window.

  • ALT + H to open "help".

  • Press ENTER on "Training".

  • DOWN ARROW to "Surf's Up".

  • Press ENTER to open.

  • Follow given instructions.
