Section 1.1 JAWS Laptop Layout

When using JAWS, you can choose between using a laptop keyboard layout or a desktop keyboard layout. The keyboard layouts are not quite the same. A laptop keyboard does not usually have a NUMPAD (number pad). So different keystrokes are used

If you have a laptop, you will be changing the keyboard layout to "laptop". The CAPSLOCK key will become a modifier for the INSERT key. This means that when INSERT is used in a keystroke, the CAPSLOCK can be used to replace the INSERT key. CAPSLOCK is often easier to reach compared with insert. You won't have to move your hands off of homerow to reach CAPSLOCK.

To use the CAPSLOCK key to capitalize all letters, press the key twice quickly. Pressing the CAPSLOCK twice quickly when JAWS is running in laptop keyboard layout will allow you to use CAPSLOCK for its original purpose.

Subsection 1.1.2 Tasks

Follow the steps below to practice important JAWS commands.

  1. First, set your keyboard layout to "laptop".

    • ALT + TAB to open the JAWS window.

    • ALT to jump to the menu bar.

    • DOWN ARROW to open the "options" menu.

    • DOWN ARROW to "basics".

    • Press ENTER.

    • TAB to "use keyboard layout".

    • LEFT ARROW or RIGHT ARROW to "laptop".

    • Press ENTER.Pressing enter activates the OK button and saves the changes.

  2. You can use keyboard help to get to know the laptop keyboard layout.

    • Press CAPSLOCK + 1. This will toggle keyboard help on or off. Try this now.

    • Practice the JAWS KEYSTROKES listed here while "keyboard help" is on to learn what the keystrokes do.

      • INSERT + M (remember to use CAPSLOCK as the modifier key if on a laptop)

      • INSERT + , (comma)

      • INSERT + . (period)

      • INSERT + J

      • INSERT + K

      • INSERT + L

      • INSERT + U

      • INSERT + I

      • INSERT + O}

    • What did these keystrokes do? The keystrokes will work any time JAWS is running.

    • Use keyboard help now to get to know the laptop layout.

    • Press the key or keystroke twice quickly to get a more detailed description of what the key or keystroke will do.

    • Now turn "keyboard help" off. Press INSERT + 1 to toggle "keyboard help" off.
